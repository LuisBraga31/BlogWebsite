# Projeto de um blog

Esse projeto foi o desenvolvimento de um simples blog utilizando o framework Tailwind CSS pela primeira vez.

## Tecnologias Utilizadas:

Essas foram as seguintes tecnologias usadas para criação do projeto.

* React
* JavaScript
* Tailwind CSS
